# 제목: 비트코인 결제와 기존 금융 지급결제의 구조적 차이 비교

## 부서명: 디지털서비스개발부
## 이름: 신승선

-------------------

## 레포트 내용
비트코인 결제와 기존 금융 지급결제의 구조적인 차이는 앞서 작성한 레포트에서 언급된 "신뢰를 보증하는 주체"와 "중개자의 유무"에서 가장 크게 나타난다고 볼 수 있다.
우선 이러한 차이점을 이해하기 위해서는 두 방식의 결제 흐름을 비교해볼 필요가 있다.
- 기존 금융 지급 결제
  1. 요청 : 구매자 > 결제대행사(PG) / VAN사
  2. 승인 : 카드사 / 발급 은행 > 신용한도 및 잔액 확인
  3. 청산 및 결제 : 중앙은행이나 금융결제원을 통해 은행간 실제 자금 이동 (보통 1~3영업일 소요)
 
- 비트코인 지급 결제
  1. 요청 : 송금자가 수신자의 '지갑주소'로 비트코인 전송 (디지털 서명)
  2. 검증 : 거래내역이 네트워크에 전파되고, 전 세계의 노드들이 거래의 유효성을 검증
  3. 확정 : 채굴자가 검증된 거래들을 모아 하나의 '블록'으로 만들고(약 10분 소요), 기존 블록체인에 연결하면 송금과 정산이 동시에 완료
 
위와 같이 중개자는 각각 '은행, 신용카드사, 결제원 등 금융기관' vs '알고리즘과 네트워크 참여자'로 차이가 있으며,
신뢰의 기반은 '기관에 대한 신뢰' vs '암호학적 증명' 으로 차이가 난다.

여기에서 파생되는 기존 금융 지급결제와 비교되는 비트코인 지급결제의 장단점이 있는데, 다음과 같이 정리할 수 있다.
장점 : 전 세계 어디로든 중개자 없이 24시간 365일 거액을 저렴하게 보낼 수 있고, 국가의 통제를 받지 않는다.
단점 : 주소를 잘못입력하거나 해킹을 당하면 누구도 책임져주지 않는다.

추가적으로, 위에서 언급된 것과 같이 비트코인 지급결제가 블록생성 시간과 동일한 시간이 소요되는 것을 해결하기 위해 "라이트닝 네트워크"라는 기술이 등장하였다.
라이트닝 네트워크란 비트코인의 Layer 2 솔루션으로, 속도를 극대화하고 트랜잭션 수수료를 최소화하고 있다.
실제로 비트코인을 법정통화로 채택한 엘살바도르에서는 해당 기술을 통해 지급결제를 제공하고 있다.
